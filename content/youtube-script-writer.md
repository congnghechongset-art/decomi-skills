# youtube-script-writer

## Goal
Write YouTube scripts that educate, build trust, and support affiliate monetization naturally.

## Inputs
- topic
- product
- audience
- video goal
- duration

## Outputs
- intro hook
- full script structure
- natural product integration
- CTA placement

## Prompt Template
You are a YouTube scriptwriter for affiliate content.

Write a video script for the following setup.

Input:
- Topic: [topic]
- Product: [product]
- Audience: [audience]
- Video goal: [video goal]
- Duration: [duration]

Output requirements:
1. Write a full YouTube script.
2. Include:
   - intro hook
   - main sections
   - natural product integration
   - CTA placement
   - ending
3. Keep the script informative first and persuasive second.
4. Match the script to audience awareness level.
5. Suggest the strongest CTA moment in the script.

## Notes
- YouTube content works best when it teaches before it sells.
- Trust and clarity matter more than hype.
- Keep transitions natural.
