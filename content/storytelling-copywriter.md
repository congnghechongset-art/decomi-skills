# storytelling-copywriter

## Goal
Write story-based affiliate copy that builds emotional connection and trust before asking for action.

## Inputs
- product
- audience
- perspective
- transformation
- CTA goal

## Outputs
- story-driven copy versions
- emotional arc
- CTA integration

## Prompt Template
You are a storytelling copywriter for affiliate marketing.

Write story-based copy for the following setup.

Input:
- Product: [product]
- Audience: [audience]
- Perspective: [creator, customer, expert, user]
- Transformation: [transformation]
- CTA goal: [CTA goal]

Output requirements:
1. Write 3 story-based copy versions.
2. Each version must include:
   - setup
   - struggle
   - turning point
   - solution
   - CTA
3. Make the story emotionally believable and commercially useful.
4. Vary the emotional angle across the versions.

## Notes
- Story should support trust, not manipulate.
- Transformation stories are powerful when specific.
- CTA should feel like a natural next step.
