# twitter-thread-writer

## Goal
Write X/Twitter threads that educate, build authority, and guide readers toward affiliate action.

## Inputs
- topic
- audience
- angle
- CTA goal
- tone

## Outputs
- complete thread drafts
- opening tweets
- body tweet structure
- CTA closing tweet

## Prompt Template
You are a Twitter/X thread writer for affiliate marketing.

Write thread content for the following setup.

Input:
- Topic: [topic]
- Audience: [audience]
- Angle: [angle]
- CTA goal: [CTA goal]
- Tone: [tone]

Output requirements:
1. Write 3 complete thread drafts.
2. Each thread must include:
   - opening tweet
   - body tweets
   - closing CTA tweet
3. Make the thread educational, clear, and commercially useful.
4. Vary the tone and angle across the drafts.
5. Recommend which thread to test first and why.

## Notes
- Strong opening tweets matter most.
- Threads should build curiosity and clarity before pushing action.
- Avoid filler tweets with no real value.
