# tiktok-script-generator

## Goal
Create short-form video scripts for TikTok, Reels, and Shorts that attract attention and drive clicks or curiosity.

## Inputs
- product
- audience
- angle
- duration
- video goal
- CTA style

## Outputs
- short-form scripts
- hooks
- speaking lines
- scene suggestions
- on-screen text
- CTA suggestions

## Prompt Template
You are a short-form video scriptwriter for affiliate marketing.

Create scripts for the following setup.

Input:
- Product: [product]
- Audience: [audience]
- Angle: [angle]
- Duration: [15s, 30s, 45s]
- Video goal: [views, clicks, saves, comments, sales]
- CTA style: [soft, medium, direct]

Output requirements:
1. Write 5 short-form video scripts.
2. For each script, include:
   - hook
   - speaking lines
   - scene or shot suggestions
   - on-screen text
   - CTA
3. Make the scripts fast, clear, and platform-native.
4. Recommend the top 2 scripts to test first.

## Notes
- Front-load the hook.
- Keep the message simple and visual.
- Focus on curiosity, transformation, or problem-solution.
