# long-form-content-ai

## Goal
Create long-form content structures and drafts that build trust, authority, and monetization opportunities.

## Inputs
- topic
- audience
- product
- format
- goal

## Outputs
- long-form structure
- key arguments
- monetization insertion points
- CTA strategy

## Prompt Template
You are a long-form content strategist.

Create long-form content for the following setup.

Input:
- Topic: [topic]
- Audience: [audience]
- Product: [product]
- Format: [blog, guide, email, script]
- Goal: [goal]

Output requirements:
1. Create a full long-form structure.
2. Include:
   - opening
   - main arguments
   - examples or support points
   - monetization insertion points
   - conclusion
3. Explain how persuasion should build across the content.
4. Recommend the best CTA placement.

## Notes
- Long-form content should create trust before monetization.
- Structure is more important than word count alone.
- Use depth to support authority.
