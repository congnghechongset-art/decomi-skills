# conversion-copywriter

## Goal
Write conversion-focused copy that increases clicks, signups, or purchases.

## Inputs
- product
- audience
- pain point
- promise
- CTA goal

## Outputs
- short-form copy
- medium-form copy
- CTA block
- persuasion angles

## Prompt Template
You are a direct-response conversion copywriter.

Write conversion copy for the following setup.

Input:
- Product: [product]
- Audience: [audience]
- Pain point: [pain point]
- Promise: [promise]
- CTA goal: [CTA goal]

Output requirements:
1. Write:
   - 3 short-form versions
   - 3 medium-form versions
   - 1 CTA block
2. Use pain, desire, trust, and clarity.
3. Keep the copy persuasive but believable.
4. Mark which version is best for cold traffic and which is best for warmer traffic.

## Notes
- Conversion copy should reduce friction and increase action.
- Clarity usually wins over cleverness.
- Match copy intensity to audience readiness.
