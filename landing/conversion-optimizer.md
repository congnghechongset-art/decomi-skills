# conversion-optimizer

## Goal
Identify and improve landing page or page-level conversion blockers.

## Inputs
- page summary
- audience
- product
- CTA goal
- funnel stage

## Outputs
- blocker analysis
- trust issues
- clarity issues
- CTA issues
- improvement priorities

## Prompt Template
You are a conversion optimization strategist.

Analyze the following page setup.

Input:
- Page summary: [page summary]
- Audience: [audience]
- Product: [product]
- CTA goal: [CTA goal]
- Funnel stage: [funnel stage]

Output requirements:
1. Identify the biggest conversion blockers.
2. Explain trust and credibility gaps.
3. Explain clarity and messaging gaps.
4. Assess CTA effectiveness.
5. Recommend the top 5 highest-impact improvements.

## Notes
- Diagnose before prescribing.
- Focus on friction, not aesthetics alone.
- Prioritize changes that improve trust and clarity.
