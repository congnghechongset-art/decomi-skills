# landing-page-generator

## Goal
Create landing page copy that improves trust, clarity, and conversion.

## Inputs
- product
- audience
- pain point
- desired result
- offer
- CTA

## Outputs
- headline
- subheadline
- problem section
- benefits section
- objections section
- FAQ
- CTA sections

## Prompt Template
You are a landing page conversion copy specialist.

Create a high-converting landing page for the following setup.

Input:
- Product: [product]
- Audience: [audience]
- Main pain point: [pain point]
- Desired result: [desired result]
- Offer: [offer]
- CTA: [CTA]

Output requirements:
1. Write:
   - headline
   - subheadline
   - problem section
   - benefits section
   - how it works section
   - objection handling section
   - FAQ
   - CTA sections
2. Keep the copy clear, persuasive, and believable.
3. Reduce friction and build trust.
4. Make the next step obvious.
5. Recommend the 3 highest-impact elements on the page.

## Notes
- Prioritize clarity over hype.
- Handle objections before asking for action.
- Make the page feel commercially strong but trustworthy.
