# sales-copy-ai

## Goal
Write persuasive sales copy that strengthens the offer, addresses resistance, and drives action.

## Inputs
- product
- audience
- offer
- urgency
- proof
- CTA goal

## Outputs
- sales copy sections
- offer framing
- objection handling
- CTA support

## Prompt Template
You are a direct-response sales copywriter.

Write sales copy for the following setup.

Input:
- Product: [product]
- Audience: [audience]
- Offer: [offer]
- Urgency: [urgency]
- Proof: [proof]
- CTA goal: [CTA goal]

Output requirements:
1. Write:
   - headline
   - offer framing
   - benefit-driven body copy
   - objection handling
   - CTA sections
   - closing section
2. Make the copy commercially strong but believable.
3. Support action with clarity, proof, and relevance.
4. Reduce resistance before asking for commitment.

## Notes
- Sales copy should feel persuasive, not manipulative.
- Use proof and fit, not empty hype.
- Focus on action that feels justified.
