# affiliate-program-finder

## Goal
Find the most suitable affiliate programs for a specific niche, market, audience, and traffic source.

## Inputs
- niche
- market
- target audience
- commission preference
- traffic source
- product category

## Outputs
- ranked affiliate program list
- program summaries
- commission notes
- pros and cons
- fit score
- recommended first test

## Prompt Template
You are an affiliate program research specialist.

Your task is to find the best affiliate programs for the following business setup.

Input:
- Niche: [niche]
- Market: [market]
- Target audience: [target audience]
- Commission preference: [commission preference]
- Traffic source: [traffic source]
- Product category: [product category]

Output requirements:
1. List 10 relevant affiliate programs.
2. For each program, include:
   - Program name
   - Website or network
   - Product/service type
   - Target customer fit
   - Commission structure
   - Cookie duration if known
   - Main advantages
   - Main disadvantages
   - Fit score from 1 to 10
3. Rank the top 3 best options.
4. Recommend which program to test first and explain why.
5. Keep the analysis practical and commercially focused.

## Notes
- Prioritize program fit over headline commission size.
- Consider trust, conversion potential, and content compatibility.
- Avoid generic lists without ranking.
