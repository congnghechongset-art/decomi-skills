# content-calendar-ai

## Goal
Build structured content calendars that support visibility, trust, and affiliate conversion.

## Inputs
- niche
- platforms
- content goals
- posting frequency
- funnel priorities

## Outputs
- calendar plan
- topic assignments
- format assignments
- CTA mix

## Prompt Template
You are a content planning strategist.

Create a content calendar for the following setup.

Input:
- Niche: [niche]
- Platforms: [platforms]
- Content goals: [content goals]
- Posting frequency: [posting frequency]
- Funnel priorities: [funnel priorities]

Output requirements:
1. Create a 30-day content calendar.
2. For each entry include:
   - topic
   - format
   - platform
   - funnel stage
   - CTA style
3. Balance awareness, trust, and conversion content.
4. Recommend what themes should repeat.

## Notes
- Calendars should be usable, not overwhelming.
- Maintain balance across funnel stages.
- Repetition is useful when done strategically.
