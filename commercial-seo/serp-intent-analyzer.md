# serp-intent-analyzer

## Goal
Analyze search intent patterns so content type and monetization approach match what users expect.

## Inputs
- keyword
- niche
- market
- monetization goal

## Outputs
- SERP intent analysis
- likely dominant content type
- monetization fit
- content recommendation

## Prompt Template
You are a SERP intent analyst.

Analyze the likely search intent for the following setup.

Input:
- Keyword: [keyword]
- Niche: [niche]
- Market: [market]
- Monetization goal: [monetization goal]

Output requirements:
1. Explain the likely dominant search intent.
2. Assess whether the query is:
   - informational
   - commercial
   - transactional
3. Recommend the best content format.
4. Explain how affiliate monetization should be handled for this intent.
5. Recommend whether this keyword should be prioritized now.

## Notes
- Intent mismatch ruins SEO and monetization.
- Commercial fit matters more than raw keyword volume.
- Use intent to choose the right content type.
