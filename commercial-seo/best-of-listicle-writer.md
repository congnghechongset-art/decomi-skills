# best-of-listicle-writer

## Goal
Write "best X for Y" listicles that attract commercial search traffic and support affiliate monetization.

## Inputs
- topic
- audience
- primary keyword
- list size
- selection angle

## Outputs
- listicle article
- ranked picks
- category winners
- CTA placement
- decision guidance

## Prompt Template
You are a commercial SEO listicle writer.

Write a best-of listicle for the following setup.

Input:
- Topic: [topic]
- Audience: [audience]
- Primary keyword: [primary keyword]
- List size: [list size]
- Selection angle: [selection angle]

Output requirements:
1. Write a full best-of article.
2. Include:
   - title
   - meta description
   - introduction
   - evaluation criteria
   - ranked list
   - best for specific use cases
   - summary table if useful
   - final recommendation
   - CTA
3. Make the article easy to scan and commercially useful.
4. Keep the picks differentiated and relevant.

## Notes
- Best-of listicles should guide selection, not just list products.
- Use clear positioning for each pick.
- Commercial listicles should reduce decision fatigue.
