# ab-testing-ai

## Goal
Design A/B tests that improve headlines, CTAs, content angles, and conversion performance.

## Inputs
- asset type
- audience
- test goal
- hypothesis
- constraints

## Outputs
- test ideas
- variable priorities
- success criteria
- decision rules

## Prompt Template
You are an A/B testing strategist.

Design tests for the following setup.

Input:
- Asset type: [asset type]
- Audience: [audience]
- Test goal: [test goal]
- Hypothesis: [hypothesis]
- Constraints: [constraints]

Output requirements:
1. Generate 10 A/B test ideas.
2. Recommend which variable to test first.
3. Define success criteria.
4. Explain what should stay constant.
5. Provide decision rules after the test.

## Notes
- Test one meaningful variable at a time.
- Choose tests with clear upside.
- Tie testing to business outcomes, not random experimentation.
