# performance-tracker

## Goal
Design a KPI tracking system for affiliate content, funnel, and campaign performance.

## Inputs
- campaign type
- platforms
- funnel stages
- main KPI
- reporting cadence

## Outputs
- KPI framework
- metric mapping
- review schedule
- alert logic

## Prompt Template
You are a performance tracking strategist.

Create a KPI tracking framework for the following setup.

Input:
- Campaign type: [campaign type]
- Platforms: [platforms]
- Funnel stages: [funnel stages]
- Main KPI: [main KPI]
- Reporting cadence: [reporting cadence]

Output requirements:
1. Define the core KPIs to track.
2. Map KPIs to funnel stages.
3. Recommend how often to review them.
4. Identify warning signals to watch for.
5. Explain how tracking should support optimization decisions.

## Notes
- Track what drives action, not vanity metrics only.
- Framework should support affiliate revenue decisions.
- Keep reporting practical and reusable.
