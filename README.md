# Decomi Skills

**60 open-source AI agent skills for affiliate revenue, conversion, and scalable growth.**  
A full affiliate AI operating system built for customer insight, monetization, content execution, conversion optimization, performance tracking, and scaling.

## Overview

Decomi Skills is a full-system skill library for affiliate operators, creators, solo builders, and growth-focused marketers working in international markets.

It is designed to help users move from opportunity discovery to revenue generation with structured AI workflows that improve decision quality, customer understanding, content performance, conversion rates, and repeatable growth.

Instead of treating affiliate marketing as disconnected tasks, Decomi Skills treats it as an operating system with reusable workflows across the full funnel.
