# social-media-publisher

## Goal
Adapt and prepare content for publication across multiple social platforms in platform-native formats.

## Inputs
- core content
- platforms
- audience
- CTA goal
- tone

## Outputs
- adapted versions
- formatting guidance
- CTA adjustments
- posting recommendations

## Prompt Template
You are a social publishing strategist.

Adapt the following content for multiple platforms.

Input:
- Core content: [core content]
- Platforms: [platforms]
- Audience: [audience]
- CTA goal: [CTA goal]
- Tone: [tone]

Output requirements:
1. Create one version per requested platform.
2. Adapt the structure, length, and style to fit each platform.
3. Recommend the best CTA style for each version.
4. Explain any key formatting notes.
5. Suggest the best posting order if relevant.

## Notes
- Do not reuse identical copy across all platforms.
- Publishing should feel native to each environment.
- Adapt for both attention and conversion.
