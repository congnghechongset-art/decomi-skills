# multi-platform-poster

## Goal
Adapt one content idea into multiple platform-native versions for wider distribution.

## Inputs
- core idea
- platforms
- audience
- CTA goal
- tone

## Outputs
- platform-specific versions
- formatting recommendations
- CTA recommendations
- launch priority

## Prompt Template
You are a multi-platform content adaptation specialist.

Adapt the following core idea into multiple platform-ready versions.

Input:
- Core idea: [core idea]
- Platforms: [platforms]
- Audience: [audience]
- CTA goal: [CTA goal]
- Tone: [tone]

Output requirements:
1. Create one version for each platform requested.
2. Adapt the language and structure to fit each platform.
3. Recommend the best CTA style for each platform.
4. Explain which platform should be launched first and why.
5. Keep each version commercially useful and audience-appropriate.

## Notes
- Do not copy-paste the same structure across platforms.
- Respect platform-native style.
- Optimize for both attention and conversion.
