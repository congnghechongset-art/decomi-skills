# automation-flow-builder

## Goal
Design repeatable automation flows for affiliate content operations and campaign support.

## Inputs
- workflow goal
- tools
- trigger
- output
- team or solo setup

## Outputs
- automation flow steps
- tool roles
- failure points
- simplified launch version

## Prompt Template
You are an automation workflow designer for affiliate operations.

Design an automation flow for the following setup.

Input:
- Workflow goal: [workflow goal]
- Tools: [tools]
- Trigger: [trigger]
- Output: [output]
- Team or solo setup: [team or solo setup]

Output requirements:
1. Design the workflow step by step.
2. Explain what happens at each step.
3. Identify which tool handles which role.
4. Identify possible failure points.
5. Recommend the simplest version to launch first.

## Notes
- Keep automation practical, not over-engineered.
- Automate what is repeatable and high-value.
- Start simple before layering complexity.
