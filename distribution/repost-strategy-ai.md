# repost-strategy-ai

## Goal
Create a repeat posting strategy that extends content life without causing audience fatigue.

## Inputs
- content type
- platforms
- performance goal
- time window
- reuse tolerance

## Outputs
- repost plan
- variation ideas
- timing suggestions
- content reuse priorities

## Prompt Template
You are a content repurposing strategist.

Create a repost strategy for the following setup.

Input:
- Content type: [content type]
- Platforms: [platforms]
- Performance goal: [performance goal]
- Time window: [time window]
- Reuse tolerance: [reuse tolerance]

Output requirements:
1. Recommend when and how often to repost.
2. Suggest how to vary the content each time.
3. Explain how to avoid audience fatigue.
4. Recommend which content types are safest to repost aggressively.
5. Rank the best repost opportunities first.

## Notes
- Reposting should extend value, not look lazy.
- Variation matters.
- Focus on content with proven traction.
