# distribution-checklist-ai

## Goal
Create publishing checklists that reduce mistakes and improve content execution quality.

## Inputs
- campaign type
- platforms
- workflow complexity
- team size
- CTA goal

## Outputs
- pre-publish checklist
- publish checklist
- post-publish checklist
- common mistake list

## Prompt Template
You are a publishing operations specialist.

Create a distribution checklist for the following setup.

Input:
- Campaign type: [campaign type]
- Platforms: [platforms]
- Workflow complexity: [workflow complexity]
- Team size: [team size]
- CTA goal: [CTA goal]

Output requirements:
1. Create a pre-publish checklist.
2. Create a publish checklist.
3. Create a post-publish checklist.
4. List the most common execution mistakes to avoid.
5. Keep the checklist practical and reusable.

## Notes
- Checklists should reduce friction and missed details.
- Include CTA, links, and formatting checks.
- Optimize for repeatable execution quality.
