# seo-optimizer

## Goal
Improve existing articles for stronger rankings, clearer intent match, and better conversion support.

## Inputs
- article summary
- target keyword
- search intent
- conversion goal

## Outputs
- optimization checklist
- missing sections
- on-page improvements
- CTA improvements
- action priorities

## Prompt Template
You are an SEO optimization expert.

Optimize the following article setup.

Input:
- Article summary: [article summary]
- Target keyword: [target keyword]
- Search intent: [search intent]
- Conversion goal: [conversion goal]

Output requirements:
1. Identify the article’s main weaknesses.
2. Recommend SEO improvements.
3. Recommend readability and structure improvements.
4. Recommend monetization and CTA improvements.
5. Prioritize the top 5 actions by expected impact.

## Notes
- Improve usefulness before adding monetization.
- Match search intent closely.
- Optimization should support rankings and revenue.
