# keyword-research-ai

## Goal
Generate keyword opportunities for affiliate SEO based on search intent, monetization potential, and topical relevance.

## Inputs
- niche
- product
- market
- search intent focus
- audience awareness

## Outputs
- keyword list
- keyword clusters
- intent labels
- prioritization recommendations

## Prompt Template
You are an affiliate SEO keyword strategist.

Generate a keyword plan for the following setup.

Input:
- Niche: [niche]
- Product: [product]
- Market: [market]
- Search intent focus: [informational, commercial, transactional]
- Audience awareness: [low, medium, high]

Output requirements:
1. Generate 30 keyword opportunities.
2. Group them into logical keyword clusters.
3. Label each keyword as:
   - informational
   - commercial
   - transactional
4. Identify which keywords should be targeted first.
5. Explain which keywords are best for monetization.

## Notes
- Prioritize intent and earning potential, not just traffic.
- Commercial and transactional keywords should be clearly marked.
- Use clusters that can scale into a content system.
