# blog-outline-generator

## Goal
Create SEO-friendly article outlines that match search intent and support affiliate monetization.

## Inputs
- keyword
- search intent
- audience
- article type
- monetization angle

## Outputs
- title options
- H2/H3 outline
- CTA placement suggestions
- structure notes

## Prompt Template
You are an SEO content outline specialist.

Create a detailed outline for the following article.

Input:
- Keyword: [keyword]
- Search intent: [search intent]
- Audience: [audience]
- Article type: [guide, review, comparison, listicle]
- Monetization angle: [monetization angle]

Output requirements:
1. Write 5 title options.
2. Create a full H2/H3 outline.
3. Recommend where to place soft CTAs.
4. Ensure the structure matches the search intent.
5. Explain what sections are most important for usefulness and conversion.

## Notes
- Build outlines that are easy to write and monetize.
- Match outline depth to search intent.
- Strong outlines reduce weak SEO writing later.
