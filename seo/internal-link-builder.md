# internal-link-builder

## Goal
Design internal linking structures that strengthen SEO authority and support affiliate conversions.

## Inputs
- site topic
- content inventory
- priority pages
- monetization pages

## Outputs
- internal link map
- anchor suggestions
- support page relationships
- linking priorities

## Prompt Template
You are an internal linking strategist for affiliate SEO.

Design an internal linking plan for the following setup.

Input:
- Site topic: [site topic]
- Content inventory: [content inventory]
- Priority pages: [priority pages]
- Monetization pages: [monetization pages]

Output requirements:
1. Recommend how existing pages should link to each other.
2. Identify the best support pages for each priority page.
3. Suggest anchor text ideas.
4. Explain how internal links should strengthen both SEO and monetization.
5. Rank the most important linking opportunities first.

## Notes
- Prioritize money pages and high-intent content.
- Internal links should help both users and search engines.
- Avoid random linking with no strategic flow.
