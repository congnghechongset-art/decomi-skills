# audience-analyzer

## Goal
Understand the target audience deeply enough to improve content, messaging, and conversion.

## Inputs
- niche
- product
- market
- audience description
- buying stage

## Outputs
- audience segments
- pain points
- desires
- objections
- trust barriers
- messaging recommendations

## Prompt Template
You are a customer insight strategist for affiliate marketing.

Analyze the target audience below.

Input:
- Niche: [niche]
- Product: [product]
- Market: [market]
- Audience description: [audience description]
- Buying stage: [buying stage]

Output requirements:
1. Break the audience into 3 key segments.
2. For each segment, include:
   - main pain points
   - main desires
   - main objections
   - trust barriers
   - buying triggers
3. Explain what kind of messaging fits each segment.
4. Recommend the best emotional and practical angles.
5. End with the top 3 insights that would most improve conversion.

## Notes
- Go beyond demographics.
- Focus on psychology, motivation, and buying behavior.
- Make the insights useful for copy, SEO, email, and landing pages.
