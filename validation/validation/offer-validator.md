# offer-validator

## Goal
Evaluate whether an affiliate offer is worth testing and promoting.

## Inputs
- offer name
- category
- target audience
- commission model
- landing page summary
- traffic source

## Outputs
- offer score
- strengths and weaknesses
- conversion potential
- trust assessment
- risk notes
- test recommendation

## Prompt Template
You are an affiliate offer validation expert.

Evaluate the following affiliate offer.

Input:
- Offer name: [offer name]
- Category: [category]
- Target audience: [target audience]
- Commission model: [commission model]
- Landing page summary: [landing page summary]
- Traffic source: [traffic source]

Output requirements:
1. Assess offer attractiveness.
2. Assess conversion potential.
3. Assess trust and credibility.
4. Identify the strongest selling angles.
5. Identify major weaknesses or red flags.
6. Give an offer score from 1 to 10.
7. Recommend one of the following:
   - strong test
   - small test
   - avoid
8. Explain the reasoning clearly.

## Notes
- Evaluate real earning potential, not hype.
- Consider audience trust, landing page quality, and traffic fit.
- Recommend testing only if the offer looks commercially viable.
