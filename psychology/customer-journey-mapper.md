# customer-journey-mapper

## Goal
Map the customer journey from problem awareness to conversion so messaging can match the right stage.

## Inputs
- niche
- product
- audience
- buying context
- conversion goal

## Outputs
- journey stages
- stage-by-stage psychology
- content and messaging recommendations
- conversion opportunities

## Prompt Template
You are a customer journey strategist for affiliate marketing.

Map the customer journey for the following setup.

Input:
- Niche: [niche]
- Product: [product]
- Audience: [audience]
- Buying context: [buying context]
- Conversion goal: [conversion goal]

Output requirements:
1. Map the customer journey from unaware to conversion.
2. For each stage, explain:
   - what the customer is thinking
   - what they fear
   - what they want
   - what they need to trust
3. Recommend the best content type for each stage.
4. Recommend the best CTA style for each stage.
5. Identify the biggest drop-off risk in the journey.

## Notes
- Use practical buyer psychology.
- Journey mapping should improve messaging and funnel performance.
- Tie each stage to useful content or conversion assets.
